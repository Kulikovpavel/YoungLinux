План исследования к проекту "Использование gameBlender в моделировании физических явлений"

Проблема
Современное программное обеспечение часто ориентировано на быструю разработку. Существуют ли среды для быстрого моделирования виртуальной реальности и насколько хорошо они моделируют физические законы реального мира?

Гипотеза
Несмотря на то, что программа Blender (среда 3D-моделирования) является программой с открытым исходным кодом и имеет небольшую историю, за счет встроенного в нее так называемого игрового движка, может использоваться в качестве среды для моделирования физического мира.

План

  1. Были изучены возможности программы Blender, как среды для создания трехмерной графики и анимации.
  2. Поскольку литература по game engine существует почти исключительно на английском языке, были частично переведены несколько источников. Изучены особенности работы с игровым движком.
  3. Изучен раздел «Механика» физики.
  4. Созданы интерактивные модели «Сила», «Эластичность», «Рычаг», «Сила – векторная величина» и другие с помощью game engine в Blender.
  5. Предприняты попытки создания сложной модели для исследования поведения реального объекта под влиянием разных сил.
  6. Сделан вывод о возможностях gameBlender для моделирования и исследования физических явлений.
