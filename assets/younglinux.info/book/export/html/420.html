Урок 3. Работа с текстом (план-конспект)

I - текст в контур, применение фильтра

  1. Создать надпись "Программирование - это процесс создания компьютерных программ".
  2. Изменить цвет, использовать два цвета.
  3. Изменить межсимвольный и межстрочный интервалы
  4. Преобразовать в контур.
  5. Вытянуть узля отдельных букв.
  6. Найти и использовать подходящий фильтр.

II - сохранение и экспорт

  1. Сохранить файл svg.
  2. Изметить размер холста так, чтобы в него помещалось только изображение (без полей, которые создает фильтр).
  3. Нарисовать прямоугольник для фона. Переместить его ниже текста.
  4. Экспортировать в растр страницу.
  5. Закрыть Inkscape.
  6. Пояснить разницу между файлами svg и png.

III - фильтр "Рассеивание->Аэрограф"

  1. Большие буквы (например, "WWW"). Размер шрифта больше 200 pt.
  2. Поискать интересный шрифт.
  3. Настроить цвет и контур.
  4. Использовать фильтр.

IV - текст по контуру

  1. Фраза "Язык программирования - это язык для записи компьютерных программ".
  2. Размер шрифта 20-24 pt.
  3. Нарисовать S-образную (по горизонтали) линию инструментом Безье.
  4. Выделить оба объекта, Текст->Разместить по контуру.
  5. У отдельных символов изменить смещение от линии шрифта и угол поворота (вращение).
  6. Раскрасить текст.
  7. Сделать пару символов намного больше остальных.
  8. Преобразовать текст в контур и к большим символам применить фильтр.
  9. (Двойной щелчек по группе, чтобы получить доступ к составляющим.)
