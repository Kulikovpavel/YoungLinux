План исследования к проекту "Исследование возможностей языка программирования Python по управлению данными"

Проблема
Решение задач системного администрирования по управлению данными требует наличие специальных программных средств. Часто системные администраторы сами пишут (создают) разнообразные скрипты (программы) для решения таких задач.

Гипотеза
Язык программирования Python является эффективным средством для решения задач управления данными: копирования, перемещения, переименования больших объемов файловых данных. Кроме того, из-за кросплатформенности языка, сценарии, написанные на данном языке, будут работать как в операционной системе Windows, так и системах на базе ядра Linux.

План

  1. Введение (модуль os, проблемы администрирования данных).
  2. Создание простейших скриптов для работы с файлами (+ их оформление в виде функций).
    a. Просмотр дерева каталога с занесением полных имен всех файлов в список (a).
    b. Поиск файлов в дереве каталога с занесением полных имен найденных файлов в список:
    i. по шаблону имени (i),
    ii. по размеру (ii),
    iii. по дате создания (iii-1) и изменения (iii-2).
    iv. дополнение: включение работы с каталогами
    c. Действия над файлами с списке (i, ii, iii):
    i. переименование файлов,
    ii. копирование и перемещение файлов,
    iii. удаление файлов.
    d. Сохранение списков в файл и их чтение оттуда
    e. Обработка ошибок
  3. Создание модуля и программы
    a. Объединение скриптов в один файл (модуль)
    b. Оптимизация модуля
    c. Документирование
    d. Диалоговый режим (программа)
    e. Возможности графического интерфейса
  4. Дополнительные возможности программы
    a. Создание каталогов
    b. Работа с ветвями (модуль shutil)
    c. Архивации данных
    d. Объекнто-ориентированный модуль (программа).
  5. Выводы (описание достигнутого и перспектив).
